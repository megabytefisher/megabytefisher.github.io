My projects

Automaton

C#, C++, Java - 2016

Automaton is a tool for automating Android apps. It is composed of two seperate projects: AutomatonHelper, which runs on the Android platform, and Automaton, which serves as a host as tool for setting up AutomatonHelper. The helper program hooks into private Android APIs that allow the app to do things like rapidly grab screenshots from the Android screen buffer or break down the on-screen UI hierarchy. Users run Automaton from a computer and supply it with C# scripts that perform whatever action the user wishes. The two programs communicate over a REST-like API.

SilentBreakpoints

C, Win32API - 2014

While analyzing and debugging a malware sample, I couldn't reliably find a way to set breakpoints that the target program did not either unset or detect. I started researching and devised a novel method of creating pseudo-breakpoints by enabling a Windows feature called "data execution prevention" and setting  pages of memory so that they were not executable. This caused an exception to be thrown when the target code was executed, allowing me to set invisble breakpoints.
